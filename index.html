HTML

  • The head element contains information about he webpage.
  • !DOCTYPE html: tells the browser what document type it should expect
  • html tag: represents the root. Inside is an attribute of lang="en"
  • The head will also contain the tags of title, style, base, link. meta, script
  • The body element represents the visible content shown to the user.
  • The body will contain the tags for h1-h6, paragraphs p, list ul ol li, img, br line breaks, a href links, footer

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files to the working branch
  • git commit -m "add message": commits to GitHub with a message. The Git repository is now up to date
  • git pull origin main: good habit ot ensure we are working with the most up to date base branch
  • git push origin branch-name: pushes the branch to GitHub

JavaScript

  • A variable (var) is a named container that allows us to store data in our code.
  • There are three types of Var: numbers, strings, booleens
  • Control flow is the order in which a computer executes code in a script.
  • Indexing starts at 0
  • "if" statements are comparisions with a command to execute after if it reaches a specific condition
  • "for" statements are loops that will run until a condition is met
  • for statements have three parts: starting point, condition, and a statement to allow the loop to increment
  • functions are repeatable code that once "declared" can be "called" upon whenever required
